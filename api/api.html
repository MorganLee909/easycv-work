API Docs for easycv.work

- - - /api/user - - -

POST

User Creation

Creates a new user. User is logged in once created.

body = {
  • firstName: (String) user first name
  • lastName: (String) user last name
  • email: (String) user email, must be unique
  • password (String) user password, minumum 10 characters
} response = User

- - - /api/login - - -

POST

User login

Logs in the user

body = {
  • email: (String) user email
  • password: (String) user password
} response = {}

- - - /api/employer - - -

POST

Employer creation

Create new employers (work history) and then adds them to the provided CV.

body = {
  • cv: (String) id of cv to add employers to
  • employers: [{
    • name: (String) employer name
    • startDate: (Date) date
    }]
} response = [Cv]
